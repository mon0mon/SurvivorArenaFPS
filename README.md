# GDQuest - Survivor Arena FPS

## Features

### Player Control

- `LMB` to Shoot enemy
- `W`, `A`, `S`, `D` to move around
- `Space` to jump

### Mob

- Mob that follow you when they spawn
- Mob spawns at dedicated spawner
- Mob have spawn and dead particles in it
